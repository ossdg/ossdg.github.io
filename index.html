Open Source

Open-source software is software whose source code is published and made available to the public, enabling anyone to copy, modify and redistribute the source code without paying royalties or fees.learn more..

Free software movement

The free software movement is a social movement with the goal of ensuring common mans' four basic freedoms: the freedom to run it, to study and change it, and to redistribute copies with or without changes. The alternative terms "software libre", "open source", and "FOSS" are associated with the free software movement. Richard Stallman founded and launched the movement in 1983 by founding the GNU Project. And founded the Free Software Foundation in 1985 to support the movement. learn more..
